# 🍳 AI Recipe Platform

An AI-powered recipe generation platform built with **Next.js**, **TypeScript**, **Strapi CMS**, **Neon PostgreSQL**, **Clerk Authentication**, and **Google Gemini AI**.

## 🚀 Features

- 🔐 User Authentication with Clerk
- 🤖 AI Recipe Generation using Google Gemini
- 🥗 Pantry Management
- 📷 Pantry Image Scanning (AI Vision)
- ⭐ Free & Pro Subscription Plans
- 💳 Clerk Billing Integration
- 📱 Responsive UI with Tailwind CSS & shadcn/ui
- ⚡ Fast Server Actions using Next.js App Router

## 🛠️ Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- shadcn/ui
- Clerk Authentication
- Google Gemini AI
- Strapi CMS
- Neon PostgreSQL

## ⚙️ Getting Started

### Install dependencies

```bash
npm install
```

### Start Frontend

```bash
npm run dev
```

### Start Backend

```bash
npm run develop
```

## 🔑 Environment Variables

Create a `.env.local` file in the frontend:

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_STRAPI_URL=http://localhost:1337
STRAPI_API_TOKEN=
GEMINI_API_KEY=
```

Create a `.env` file in the backend with your Strapi and Neon PostgreSQL credentials.
